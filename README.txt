Installation
------------

1) Votre base de donnée eXist doit impérativement tourner sur localhost et utiliser le port 8081 (la configuration du port eXist se trouve dans /eXist/tools/jetty/etc/jetty.xml)

2) Dans l'outil d'administration eXist, créez une collection que vous nommerez twdb

3) Dans la collection twdb crée, chargez la totalité des fichiers du dossier eXist.

4) Déployez TWDB.war sur votre moteur Tomcat.
